## 1. enum (열거형)

> enumerated type<br>
> 관련된 상수 값들을 그룹핑 즉 집합하는 것<br>
> 문자형과 숫자형 기반의 열거형을 지원함<br>
> 속성을 선언해서 타입으로도 쓸 수 있고 배열로 전환해서도 쓸수 있고 그 외의 기능들로 사용 가능
